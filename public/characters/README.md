# Character Image Rules (MVP)

`/todo`의 아바타는 아래 경로 규칙을 사용합니다.

## Core

경로: `public/characters/core`

- `octopus-mono.png`
- `worry-mono.png`
- `boar-mono.png`
- `lizard-mono.png`
- `whale-mono.png`
- `ghost-mono.png`

## Method

경로: `public/characters/method`

- `robot-mono.png`
- `sparrow-mono.png`

## Optional Color Variant

같은 이름으로 `*-color.png`를 추가할 수 있습니다.  
예: `octopus-color.png`, `robot-color.png`

이미지가 없으면 UI는 fallback 텍스트(축약 라벨)로 표시됩니다.
